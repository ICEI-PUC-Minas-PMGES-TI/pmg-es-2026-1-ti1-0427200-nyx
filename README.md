# Valkyria

O projeto Valkyria surge como um santuário digital dedicado exclusivamente às mulheres apaixonadas pelo universo gamer, combatendo diretamente a toxicidade e os preconceitos que muitas vezes tornam o ambiente online hostil. 

Nossa missão é construir um ecossistema onde a segurança e o respeito são os pilares fundamentais, permitindo que cada jogadora possa expressar sua identidade, compartilhar suas conquistas e explorar novos títulos sem o receio de julgamentos ou assédios.
